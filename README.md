# 📖 [TDD-Project] <a name="about-project"></a>

> TDD-Project is a project built using the Test-Driven Development approach. Using a Solver class, we return factorials, reverse strings and incorporate a "fizzbuzz" method.

## 💻 Getting Started <a name="getting-started"></a>

> This is how a new developer could make use this project.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [ ] A PC or Mac.
- [ ] An active internet connection.
- [ ] Some knowledge of git and github.

### Setup

Clone this repository by using the command line : git clone https://github.com/asifaminisonline/TDD-Project.git

### Install

Install this project with:

bundle install

### Usage

- [ ] First you need to download this repo or clone this repo.
- [ ] Open this project in your local machine.
- [ ] Open Terminal in your machine.
- [ ] run bundle install in your terninal to install necessary required dependencies for this project.
- [ ] ruby main.rb
- [ ] now the project will be running.
